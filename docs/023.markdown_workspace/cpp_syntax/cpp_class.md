### 1.默认构造函数
* `定义`：  
  默认构造函数是`不需要提供实参`就可以调用的构造函数。
* 语法  
  (1) 类名(形参列表可选);  
  (2) 类名(形参列表可选) 函数体  
  (3) 类名() = default;  
  (4) 类名(形参列表可选) = delete;  
  (5) 类名::类名() 函数体
  (6) 类名::类名() = default;
  1）默认构造函数声明。
  2-4）~定义。
  5、6）类定义外的~定义（必须有声明）
  **形参列表：`所有形参都有默认实参`(形参包除外)**
* 隐式声明的~
* 隐式定义的~
* 弃置的~
