## 最基本的
  |                 |                    |
  | :-------------- | -----------------: |
  | Ctrl+Shift+P/F1 |           命令面板 |
  | Ctrl+P          | 快速打开，转到文件 |
  | Ctrl+Shift+N    |           新建窗口 |
  | Ctrl+Shift+W    |           关闭窗口 |
  | Ctrl+，(逗号)   |               设置 |
  | Ctrl+K Ctrl S   |         快捷键设置 |
## 基本的编辑

  |                   |                        |
  | :---------------- | ---------------------: |
  | win共通的：       |                        |
  | Ctrl+Z            |                   撤销 |
  | Ctrl+X            | 剪切(光标不选中删除一行) |
  | Ctrl+C            |                   copy |
  | Ctrl+V            |                  paste |
  | Ctrl+up/down      |              上/下滚动 |
  | Alt+pgup/pgdn     |          上/下滚动一页 |
  | 行编辑：          |                        |
  | Alt+up/down       |             上下移动行 |
  | Shift+Alt+up/down |             上下复制行 |
  | Ctrl+Enter        |             行下面插入 |
  | Ctrl+Shift+Enter  |             行上面插入 |
  | Home/End          |          转到行首/行尾 |
  | Ctrl+Home/End     |      转到文件首/文件尾 |
  | Ctrl+【/】        |        减少/增加行缩进 |
  | 额外的：          |                        |
  | Ctrl+K Ctrl+【/】 | 折叠/展开所有子/次区域  |
  | Ctrl+Shift+【/】  |          折叠/展开区域 |
  | Ctrl+K Ctrl+0/J   |      折叠/展开所有区域 |
  | Ctrl+K Ctrl+C/U   |          添加/删除注释 |
  | Ctrl+/            |             切换行注释 |
  | Shift+Alt+A       |             切换块注释 |
  | Alt+Z             |               切换换行 |
