## 文本编辑

  |             |               |
  | :---        |         ---:  |
  | F1          | 帮助与说明     |
  | F2          | 重命名         |
  | F3          | 搜索           |
  | F4          | 地址栏         |
  | F5          | 刷新           |
  | F6          | 循环宽口/桌面元素         |
  | F10         |  菜单栏/ + shift = menu键 |
  | F11         | 全屏化         |
  | F7 F8 F9 F10 F12| 应用自定义     |
  | Alt+Esc     | 在打开的应用之间切换      |
  | Alt+PageUp/PageDown | 翻页      |
  | Ctrl+Shift+Esc | 任务管理器       |
  | Shift+F10      | menu菜单       |
  | Shift+arrows   | 多选           |
  | Shift+Delete   | 彻底删除，不放到回收站  |
