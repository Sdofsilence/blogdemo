# 英语词性(part of speech)
## 实词
* **名词** noun  
  名词在句子中可以作`主语、宾语、表语、补足语、同位语、状语、定语、称呼语`等。

* **代词** pronoun  
  代词是用来代替名词、名词性短语、分句、句子的词，  
  分为九类：`人称代词、物主代词、反身代词、指示代词、疑问代词、相互代词、关系代词、连接代词和不定代词`
  * 人称代词 ...
  * 物主代词 ...
  * 反身代词 ...
  * 相互代词  
    each other('s所有格): 两者之间  
    one another('s所有格): 三者及以上
  * 指示代词  
    用来指代人或物，this,that,these,those;such,so,the same.在句子中充当主语、宾语、表语、定语等。
  * `疑问代词`：  
    * 指人  
      `who`: 主、宾、表语  
      `whom`: 宾语  
      `whose`:主、宾、表、定语  
    * 指物  
      `which`:主、宾、定语  
      `what`:主、宾、表、定语  

    whom/which 可以跟 of短语。
  * `关系代词`：引导定语从句，在从句中作主、宾、表、定语，代表所修饰的名词或代词。  
    有：`who,whom,whose,which,that,as`.
  * `连接代词`：引导名词性从句。  
    有：`who,whom,whose,which,what,whoever,whomever,whichever,whatever`.
  * 不定代词 ...
* **动词** verb
  * `实意动词`：具有完整意义，能单独作谓语。  
    * `不及物动词`：不能跟宾语。  
    * `及物动词`：能跟宾语。  
      1.跟单宾语；2.跟双宾语；3；跟复合宾语(宾语+宾语补助语)

    根据动作是否具有延续性，分为 *延续性动词* 和 *非延续性动词* 。
  * `连系动词`：本身有词义，不能单独作谓语。  
    * 状态连系动词：be
    * 持续连系动词：keep保持,rest仍然,remain保持不变，stand位于
    * 表象连系动词：seem好像，appear好像，look看起来像
    * 感官连系动词：feel,smell,sound,taste,look
    * 变化连系动词：becone变得，grow逐渐变得，turn变得，fall变得，get变得，come变得，run变得
    * 终止连系动词：prove证实，turn out变成
  * 助动词：本身无词义，不能单独作谓语。和实意动词一起构成各种时态、语态、语气及否定和疑问结构。  
    有：be,do(does,did),have(has,had),shall(should),will(would)等。
  * 情态动词：本身具有一定的词义，不能单独作谓语。  
    有：can(could),may(might),must,shall,should,will(would),need,dare,used to,had better等。
* **副词** adverb  
  表示行为或状态特征的词，修饰动词、形容词、副词、句子等，表示时间、地点、程度等。
  * 时间  
  * 地点  
  * 方式  
  * 程度  
  * 频率  
  * `疑问副词`：`when,where,how,why`,+(how long,how soon,how often,how far ...)
  * `关系副词`：`when,where,why`
  * `联系副词`：一类引导从句/不定式：`when,where,how,why`; 另一类连接句子：therefore,besides,otherwise,however,moreover,still,thus,meanwhile等。hence,thereupon,consequently,also。
* **形容词** adjective  
  描述人或物的性质、特征、状态，修饰名词和部分代词。
  * **形容词和副词句法功能**  
    * 作定语：形容词作定语位于修饰词前；副词作定语位于修饰词后。
    * 作状语：形容词作状语用于说明主语的情况，主要表示原因、结果、方式等，可位于句首、句中、句尾；  
      副词作状语，修饰动词、形容词、其他副词和句子。
    * 做表语：形容词作表语常位于系动词be,feel,get,turn,become,prove,seem等之后；副词作表语，表示方位、方向、动作、状态等。
    * 作补足语：形容词和副词都可以作主语和宾语补足语，说明主语、宾语的位置、状态、性质、特征等。

## 虚词
* **冠词** article
  * 不定冠词：表示泛指  
    `a`:  用于以 `辅音/半元音音素` 开头的词。  
    `an`: 用于以 `元音音素` 开头的词。
  * 定冠词：  
    `the`: 1.上文提到的人或物；2.世界上独一无二的/发明创造；3.最高级前；4.乐器前；  
    5.the+姓氏s(全家人/夫妻俩) the+可数名词单数(一类人或物) the+一些形容词/分词(一类人或物)  
    6.用在表 方位、地点、时间的名词前；7.用在海洋、河流、山脉、群岛、国家、党派等专有名词前  
    7.the + 比较级...the + 比较级；8.the + 年代s/年代's 表...年代/世纪；9.by the + 计量单位 表 按...  
    10.动词+sb+介词+the+身体部分
  * 零冠词  
    1.专有名词、物质名词、抽象名词表概念时；2.星期、月份、季节、节日；3.学科、三餐、语言、棋类、球类运动等；and so on...
* **介词** preposition  
  介词用来表达词与词、词与句之间关系的词。不能在句中单独作成分，后面一般有名词、代词或相当于名词的其他词、短语、从句作他的宾语。介词和它的宾语构成介词短语，在句中作状语、表语、补足语或定语等。  
  * `介词的句法功能`
    * 作状语：介词短语可以表示时间、地点、条件、原因等，修饰动词或句子，放在句末或句首。
    * 作表语：介词短语可以放在连系动词后作表语。
    * 作补足语
    * 作定语：介词短语作定语常位于修饰词后。
    * 作独立成分
* **连词** conjunction  
  在单词和单词之间、短语和短语之间、从句和从句之间起连接作用的词叫作连词。
  * `并列连词`：
    * 并列:and,both...and...,not only...but also...,neither...nor...
    * 选择:or,not...but...,whether...or...,either...or...
    * 转折:but,while,yet
    * 原因:for
  * `从属连词`；引导名词性从句或状语从句。  
    * 一类引导名词性从句：that,whether,if,as if,as though;  
    * 另一类引导状语从句：whether...or...,as if,when,while,as,before,after,till/until,even if,in case,so that,now that,as long as,so...that...,unless,once 等。
* **叹词** interjection
