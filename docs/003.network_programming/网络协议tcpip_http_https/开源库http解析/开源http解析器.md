# http解析

* http-parser[nodejs下的不再维护转到了llparse](https://github.com/nodejs/http-parser)

* llhttp[还在维护中的nodejs下的http parser](https://github.com/nodejs/llhttp)]

* httpParser[一个c++ toy http parser](https://github.com/yuesong-feng/httpParser) 单头文件的简单实现

* httpparser[Simple and fast HTTP request, response and urls parser written in C++.](https://github.com/nekipelov/httpparser) 同样是个仅头文件的简单的实现
