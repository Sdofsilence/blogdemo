# c++各个版本关于模板和泛型的变化

* ai回答不一定靠谱

|版本	|关键特性	|局限性|
|:---|:---|:---|
|C++98/03	|类模板、函数模板、模板特化、静态多态、静态断言	     |错误信息难以理解，缺乏类型约束  |
|C++11	    |auto、decltype、constexpr、模板别名	          |类型约束不足 |
|C++14	    |泛型 Lambda、改进的 constexpr、简化模板元编程	   |类型处理依然复杂 |
|C++17	    |类型处理工具、折叠表达式、std::variant	           |模板特化依然复杂 |
|C++20	    |概念（Concepts）、范围库、泛型增强、编译期数据结构	 |进一步提升灵活性，难点在于代码复杂度和学习曲线 |
