# c++移动构造函数 与 移动赋值运算符的写法


### 分两种，被移动的数据本身是否支持移动

1. 支持stl的类如vector、unique_ptr...

```
class CMoveable{
private:
    std:vector<int> data;
public:
    //移动构造
    CMoveable(CMoveable& other) noexcept : data(std::move(other.data)){};
    //移动赋值运算符
    CMoveable& operator=(CMoveable&& other) noexcept : {
        if(this != &other)
            data = std::move(other.data);
        return *this;
    }
}
```

2. 不支持移动的裸指针、内置类型、自定义类型

```
class CMoveable{
private:
    int* data;
    int size;
public:
    //移动构造
    CMoveable(CMoveable& other) noexcept : data(other.data),size(other.size) {
        other.data = nullptr;
        other.size = 0;
    };
    //移动赋值运算符
    CMoveable& operator=(CMoveable&& other) noexcept : {
        if(this != &other){
            data = other.data;
            size = other.size;
            other.data = nullptr;
            other.size = 0;
        }
        return *this;
    }
}
```
