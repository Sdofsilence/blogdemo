# c++stl容器库初始化和赋值参数总结
## 容器总结
### 序列容器
* array vector deque list forward_list
### 关联容器(有序)
* set map multiset multimap(unordered_[]) 
### 容器适配器
* stack queue priority_queue(初始化独特)

## 初始化总结
* 模版参数： **序列容器**只有\<T,分配器\> array除外\<T,size_t N\>; **关联容器**set\<T,compare,分配器\> map\<K,T,compare,分配器\>; **容器适配器**stack和queue\<T,Contariner\>; priority_queue\<T,Container,compare\>; **一句话就是，关联容器需要比较器\<comapre\>,适配器需要容器\<Container\>,priority_queue同时需要容器和比较器，它们的顺序是: 非适配器：\<T,compare,分配器\> 适配器: \<T,Container\<T\>,compare\>但是一般默认不改变第二个参数stack和queue是dueue\<T\>,priority_queue是vector\<T\>**
* 分配器：除适配器外的所有容器的初始化都可以设置分配器，而适配器可以设置容器，间接设置分配器,c++17之前的默认分配器是 std::allocator<T>,c++17起是 std::pmr::polymorphic_allocator<T>

## 序列容器初始化与赋值
### 构造函数
* (count) (count,value) **仅序列容器**
* (inputit first,inputit last)
* (const contariner&)复制构造 (container&&)移动构造
* (std::initializer_list<T> init)

### assign成员函数(仅序列容器)
* (count,value)
* (unputit first,inputit last)
* (std::initializer_list<T> init)

### operator=
* 复制构造 移动构造
* (std::initializer_list<T> init)

## 关联容器初始化和赋值
### 构造函数 
* (compare&) 构造空的容器
* (inputit first,inputit last,[compare&])
* 复制构造 移动构造
* (std::initializer_list<T> init,[compare&])

### operator=
* 复制构造 移动构造
* (std::initializer_list<T> init)

## 无序关联容器初始化和赋值
### 构造函数
* ([bucket_count][hash&][key_equal&])
* (first,last,[bucket_count][hash&][key_equal&])
* 复制构造 移动构造
* (std::initializer_list<T> init,[bucket_count][hash&][key_equal&])

### operator=
* 复制赋值 移动赋值
* (std::initializer_list<T> init)

## 容器适配器初始化和赋值
* stack queue 底层默认是deque priority_queue默认是vector 比较器默认是std::less<T>
### 构造函数
* (const container&/container&&) 复制/移动构造底层容器 (priority_queue,第一个参数为const compare&)
* (inputit first,inputit last,const compare&,container&/&&) 仅priotity_queue适配器有
* 复制构造 移动构造

### operator=
* 复制赋值 移动赋值
