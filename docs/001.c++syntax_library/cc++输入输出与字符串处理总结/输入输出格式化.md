## 不带参数的操纵器

**头文件<ios>**

下面的都可以加前缀`no`
* boolalpha   显示布尔值
* showbase      显示基数
* showpoint     显示小数点
* skipws    忽略空白符
* uppercase     大写
* unitbuf   输出缓冲区
* internal/left/right   输出对齐方式
* dec/hex/oct   输出进制
* fixed/scientific/hexfloat/defaultfloat    浮点格式

**头文件<istream>**

* ws 读取空白符

**头文件<iomanip>

* setbase(int n) 8/10/16 除此之外的 base 值会将 basefield 重置为零，这对应于十进制输出和前缀依赖的输入。
* setfill(char c) 设置填充字符
* setprecision(int n) 
* setw(int n)
* quoted(字符串)

## 输入格式化总结

| 需求 | 代码 |
|------|------|
| 读十六进制 | `std::cin >> std::hex >> x;` |
| 读含空格字符串 | `std::getline(std::cin, str);` |
| 读单个空格 | `std::cin >> std::noskipws >> c;` |
| 安全读 C 风格字符串 | `std::cin >> std::setw(N) >> buf;` |
| 读 `"true"`/`"false"` | `std::cin >> std::boolalpha >> b;` |
