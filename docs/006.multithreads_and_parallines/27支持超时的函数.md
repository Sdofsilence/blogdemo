# c++标准中支持超时的函数
| 类型/命名空间 | 函数 | 返回值 |
| :--- | :--- | :--- |
| std::this_thread | sleep_for(duration)<br>sleep_until(time_point) | n/a |
| std::condition_variable_[any] | wait_for(lock,duration)<br>wait_until(lock,time_point) | std::cv_status::timeout<br>std::cv_status::no_timeout |
|  | wait_for(lock,duration,pred)<br>wait_until(lock,duration,pred) | bool：唤醒时pred的值 |
| std::time_mutex<br>std::recursive_timed_mutex | tyr_lock_for(duration)<br>try_lock_until(time_point) | bool:true如果获取了锁，否则 false |
| std::unique_lock<timedlockable> | unique_lock(lockable,duration)<br>unique_lock(lockable,time_point) | bool:true如果获取了锁，否则 false |
|  | try_lock_for(duration)<br>try_lock_until(time_point) | bool:true如果获取了锁，否则 false |
| std::future<valuetype><br>std::shared_future<valuetype> | wait_for(duration)<br>wait_until(time_point) | std::future_status::time_out/ready/deferred |
