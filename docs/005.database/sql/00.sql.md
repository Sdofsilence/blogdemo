## mysql书籍

* 一、《MySQL 必知必会》
  《MySQL 必知必会》是指由奥瑞利出版社出版的一本介绍MySQL数据库管理系统的入门书籍。这本书详细讲解了SQL语言的基础知识，以及如何使用MySQL进行数据检索、排序、过滤和分组等操作。具体来说，它包括了以下内容：

  * 基础知识：介绍数据库和SQL的基本概念，如表、列、行、主键等。
  * MySQL简介：解释了MySQL作为客户机-服务器软件的特性，以及不同版本的差异。
  * SQL语法：从简单的SELECT语句开始，逐步讲解如何检索、排序和过滤数据。
  * 高级功能：涉及联结查询、子查询、正则表达式搜索、全文本搜索、存储过程、游标、触发器和表约束等高级主题。
  * 实用工具：介绍了MySQL命令行实用程序、MySQL Administrator和MySQL Query Browser等工具的使用方法。

这本书注重实用性，通过大量的示例和练习，帮助读者掌握MySQL的使用技巧，适用于软件开发和数据库管理人员学习参考。

---

* 二、《MySQL 是怎么运行的》

  * 这本书含有很多图，是小白学习 MySQL 底层知识的最佳书籍。对MySQL的底层运行原理进行了介绍，内容涵盖了使用MySQL的同学在求职面试和工作中常见的一些核心概念。
* 三、《高性能 MySQL》

  * 是高阶学习书籍，内容覆盖MySQL系统架构、设计应用技巧、SQL语句优化、服务器性能调优、系统配置管理和安全设置、监控分析，以及复制、扩展和备份/还原等高级主题。
