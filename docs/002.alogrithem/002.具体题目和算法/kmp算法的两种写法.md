# kmp算法的两种写法

## 简单好理解的写法：适合面试手写

* 用特殊字符 把模式串+‘#’+匹配串连接起来，求整个串的pi值（最长真前后缀）
从匹配串的开始位置检查，长度为模式串的长度的位置是匹配成功的结尾位置。-长度+1就是匹配成功位置。

## 传统的next数组写法

* 先求出来匹配串的next（pi）数组，然后双指针便利匹配串ptr1指向匹配串ptr2指向模式串，匹配就ptr1++,ptr2++;并检查长度是否为模式串的长度，是则返回ptr1-ptr2，并跳过ptr2 = = next[ptr2-1]; 匹配失败ptr1++,如果ptr2 != 0，则跳过 ptr2 = next[ptr2-1];
