# 有两种分区的算法一种是摩尔的，一种是lomuto的[geeksforgeeks](https://www.geeksforgeeks.org/dsa/hoares-vs-lomuto-partition-scheme-quicksort/)

摩尔的太复杂不适合记忆和快速写出来，所以采取lomuto的算法[wikepedia quicksort 两种分区方案](https://en.wikipedia.org/wiki/Quicksort)

Lomuto 的分区方案：

该算法的工作原理是假设枢轴元素为最后一个元素。如果任何其他元素被指定为枢轴元素，则首先将其与最后一个元素交换。现在将两个变量 `i`初始化为 `low`，`j` 也为 `low`，迭代数组并在 `arr[j] <=/< pivot `时递增 `i`，并将 `arr[i]` 与 `arr[j]` 交换，否则仅递增 `j`。从循环中出来后，将 `arr[i]` 与 `arr[hi]` 交换。返回 `i`。

**需要注意的一点是如果使用stl中的迭代器，那么hi是不包含元素的，所以hi-1是最后一个元素，下面的代码是c的方式**

```
# 好理解的分区方案
partition(arr[], lo, hi) 
    pivot = arr[hi]
    i = lo     // place for swapping
    for j := lo to hi – 1 do
        if arr[j] <= pivot then
            swap arr[i] with arr[j]
            ++i // i是下一个写入位置，边界情况是等于hi（即pivot）
    swap arr[i] with arr[hi]
    return i
```

## 两个模板一个是需要自己写parttion的，一个是允许使用std::partition

### 自己写partition

```c++
#include <iostream>
#include<vector>
#include<algorithm>

using namespace std;

vector<int>::iterator mypartition(vector<int>::iterator l,
                                  vector<int>::iterator r) {
    auto left = l, ptr = l;
    for (; ptr < r - 1; ++ptr) {
        if ((*ptr) <= *(r - 1)) {
            iter_swap(left++, ptr);
        }
    }
    iter_swap(left, r - 1);
    return left;
}

void myquicksort(vector<int>::iterator l, vector<int>::iterator r) {
    if (distance(l, r) < 2)return;
    auto p = mypartition(l, r);
    myquicksort(l, p);
    myquicksort(p + 1, r);
}

int main() {
    int n;
    cin >> n;

    vector<int> numbers;
    numbers.reserve(n);
    int tmp;
    for (int i = 0; i < n; i++) {
        cin >> tmp;
        numbers.push_back(tmp);
    }

    myquicksort(numbers.begin(), numbers.end());

    numbers.erase(std::unique(numbers.begin(), numbers.end()), numbers.end());

    for (auto& i : numbers) {
        cout << i << endl;
    }
    return 0;
}
```

### 允许使用 std::partition

```c++
#include <iostream>
#include<vector>
#include<algorithm>

using namespace std;

//先固定第一个数，然后partition剩余元素，再交换第一个数和partition_point p,p-1就是中间元素左边小，右边大于等于（等于也可以放左边）
void myquicksort(vector<int>::iterator l, vector<int>::iterator r) {
    if (distance(l, r) < 2)return;
    auto p = std::partition(l+1, r, [&l](int& item) {
        return item < *l;
        });
    std::iter_swap(l, p-1);//p-1是中间值
    myquicksort(l, p-1);
    myquicksort(p, r);
}

int main() {
    int n;
    cin >> n;

    vector<int> numbers;
    int tmp;
    for (int i = 0; i < n; i++) {
        cin >> tmp;
        numbers.push_back(tmp);
    }

    myquicksort(numbers.begin(), numbers.end());

    numbers.erase(std::unique(numbers.begin(), numbers.end()), numbers.end());

    for (auto& i : numbers) {
        cout << i << endl;
    }
    return 0;
}
```
