# 最小覆盖子串

* 知识点：哈希 双指针 字符串
* [leetcode题解](https://www.bilibili.com/video/BV1aK4y1t7Qd) 重点是理解题目中的编辑距离思想来用一个int distance记录一个字符串是否覆盖了另一个字符串的所有字符（可能有重复字符）

## 题目描述

给出两个字符串 s 和 t，要求在 s 中找出最短的包含 t 中所有字符的连续子串。

数据范围：0≤∣S∣,∣T∣≤100000≤∣S∣,∣T∣≤10000，保证s和t字符串中仅包含大小写英文字母。

要求：进阶：空间复杂度  O ( n ) O(n) ， 时间复杂度  O ( n ) O(n)例如： S = " X D O Y E Z O D E Y X N Z " S="XDOYEZODEYXNZ" T = " X Y Z " T="XYZ" 找出的最短子串为 " Y X N Z " "YXNZ".  注意： 如果 s 中没有包含 t 中所有字符的子串，返回空字符串 “”； 满足条件的子串可能有很多，但是题目保证满足条件的最短的子串唯一。

## 代码

```c++
//这个函数的复杂度是o(map.size()) 由于map.size()的值是有限的可以看作是o(1)
    //还可以用编辑距离的概念用distance来描述两个字符串是否包含了所有的字符。
    bool check(unordered_map<char,int>& map,unordered_map<char,int>& window_map){
        for(auto& pair : map){
            if(window_map.find(pair.first) == window_map.end() || window_map[pair.first] < pair.second) return false;
        }
        return true;
    }

    string minWindow(string S, string T) {
        unordered_map<char,int> map;
        unordered_map<char,int> window_map;
        for(auto& ch : T){
            map[ch] += 1;
        }

        int min_len = 10001;
        string str;
        int left = 0,right = 0;

        for(right = 0; right < S.size();){
            if(map.find(S[right]) != map.end()) window_map[S[right]] += 1;
            if(!check(map,window_map))
            {
                right++;
                continue;
            }
            //否则的话check() == true
            while(check(map,window_map)){
                if(map.find(S[left]) != map.end()) window_map[S[left]] -= 1;
                left++;//可能出现left>right
            }
            if(right - left + 2 < min_len){
                min_len = right - left + 2;
                str = S.substr(left-1,min_len);
            }
            right++;
        }
        return str;
    }
```
