# leetcode 316.去除重复字母

* 示例：输入：s = "cbacdcbc", 输出："acdb"

* 思路：单调stack:如果当前元素小于栈顶元素，且栈顶元素在后面还会出现，则出栈，直到当前元素小于栈顶元素或者栈为空。（首先遍历字符串获取26个字母的最后一次出现位置，就可以知道是否是最后一次出现）

* 具体解题见leetcode视频题解

## 大致代码框架

```c++

int lastIndex[26];
for (int i = 0; i < s.size(); i++) {
    lastIndex[s[i] - 'a'] = i;
}

stack<char> st;
for (int i = 0; i < s.size(); i++) {
    if (st.empty() || s[i] < st.top()) {
        while (!st.empty() && s[i] < st.top() && lastIndex[st.top() - 'a'] > i) {
            st.pop();
        }
        st.push(s[i]);
    }
}

string res = "";
while (!st.empty()) {
    res = st.top() + res;
    st.pop();
}
return res;

```
