# 回溯算法 ：组合子集 排列

* 分为三种情况：
  * 元素无重复不可重复选
  * 元素重复不可重复选
  * 元素无重复可重复选

## 元素无重复不可重复选

* 组合+子集 问题
  * 使用for(int i = start; i < n; i++) ; backtrace(arr,i+1);来保证元素在原始数组中的位置的顺序性
* 排列 问题
  * 使用used[i]来保证元素在回溯过程中不会出现重复选择

## 元素重复不可重复选

* 组合+子集 问题
  * 排序 + 剪枝 if( num[i] == num[i-1] ) continue; backtrace(arr,i+1);来跳过重复的元素。
* 排列 问题
  * 排序 + 剪枝 if( i > 0 && num[i] == num[i-1] && used[i-1] == false ) continue; 来保证相同的元素在排列中的相对位置不变，只有前面的相同元素被选择后，后一个相同元素才能被选择。

## 元素无重复可重复选

* 组合+子集 问题
  * 在递归的地方 backtrace(arr,i); 注意这里要是i，而不是i+1；保证了同一个元素可以重复选
* 排列 问题
  * 最简单的全排列：什么都不做，每次都for(int i = 0; i < n; i++) backtrace(arr);
