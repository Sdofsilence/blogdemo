# std::next_permutation的实现原理

next_permutation 是一个用于生成给定序列的下一个字典序排列的算法。它在全排列问题中非常常用。其基本思想是找到当前排列的下一个排列，使其在字典序上比当前排列大，但增大的幅度最小。

## 实现步骤

1. 从后向前找到第一个升序对：找到第一个满足 a[i] < a[i+1] 的位置 i。如果不存在这样的 i，说明当前排列已经是最大的排列，直接返回 False。

2. 从后向前找到第一个大于 a[i] 的元素：找到第一个满足 a[j] > a[i] 的位置 j。

3. 交换 a[i] 和 a[j]：交换这两个元素，使得排列变大。

4. 反转 i 之后的元素：将 i 之后的元素反转，使其成为最小的排列。

## 解释

1. 找到第一个升序对：从后向前遍历数组，找到第一个满足 a[i] < a[i+1] 的位置 i。例如，对于排列 2431，我们找到 2 和 4。

2. 找到第一个大于 a[i] 的元素：继续从后向前遍历数组，找到第一个大于 a[i] 的元素 a[j]。例如，对于排列 2431，我们找到 3。

3. 交换 a[i] 和 a[j]：交换这两个元素，使得排列变大。例如，交换 2 和 3，得到 3421。

4. 反转 i 之后的元素：将 i 之后的元素反转，使其成为最小的排列。例如，将 421 反转，得到 3124。
