# 编辑距离求解

* [编辑距离 leetcode题解](https://www.bilibili.com/video/BV1ea4y147FK)
* 编辑操作有三种：插入、删除、替换
* 原理(递归思路)：假设两个字符串为s1和s2，s1为源串，s2为目标串 distance为结果 三种情况：
  * 如果s1[i]==s2[j]，则不需要进行操作，distance = distance(i-1,j-1);
  * 如果s1[i]!=s2[j]，则需要在s1上面进行一次操作（插入distance(i,j-1)+1，删除distance(i-1,j)+1或替换distance(i-1,j-1)+1），distance = 1+min(distance(i,j-1),distance(i-1,j),distance(i-1,j-1));

## 递归算法

```c++
int minDistance(string_view word1, string_view word2) { 
    if(word1.size() ==0 || word2.size() == 0){
        return max(word1.size(),word2.size());
    }
    if(word1.back() == word2.back()){
        return minDistance(word1.substr(0,word1.size()-1),word2.substr(0,word2.size()-1));
    }
    return 1+min(minDistance(word1.substr(0,word1.size()-1),word2),  minDistance(word1,word2.substr(0,word2.size()-1)),
    minDistance(word1.substr(0,word1.size()-1),word2.substr(0,word2.size()-1)));
}
```

## 动态规划算法

* 由递归算法可知：
* 状态定义：`dp[i][j]为s1[i] 和 s2[j]` 的编辑距离
* 初始化：i 或者 j 为 零的时候 `dp[i][j] = i,j` 中不为零的值;
* 状态转移：`if s1[i] == s2[j] dp[i][j] = dp[i-1][j-1]  else dp[i][j] = 1+min(dp[i][j-1],dp[i-1][j],dp[i-1][j-1])`;
* 最终状态：`distance = dp[m][n]`
* 编辑过程：可以通过dp矩阵推导出来，起始点为`dp[m][n]` 终止点为 `dp[0][0]` 需要考虑边界不是每次都有三种可能。当i,j其中一个为零的时候，可以直接得出需要插入i或j个相应位置的字符。
