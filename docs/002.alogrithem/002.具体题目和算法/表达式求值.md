## 逆波兰表达式（后缀表达式求值）

* 中缀表达式转后缀表达式算法
  * 参考b站视频<https://b23.tv/yEIOemT>
  * 规则：
    * 1数字直接输出到后缀表达式
    * 2左括号（直接进栈，右括号）不进栈，让栈顶的元素和（比较，如果不是，出栈，是也出栈，但是直接break；
    * 3+-*/算数运算符，当扫描到的是算术运算符，如果栈为空直接入栈，栈不为空，那么和栈顶元素比较，如果当前运算符优先级小于等于栈顶运算符优先级，栈顶元素出栈，输出到后缀表达式中，直到条件不满足或者栈为空，然后当前运算符入栈。

* 后缀表达式求值算法
  * 遍历后缀表达式，如果是数值直接入栈，如果是符号出栈，然后再出栈两个数值作为两个操作数，计算结果后，把结果入栈；直到遍历结束后，栈中只有一个数值，就是表达式的结果。

### deepseek回答

Pratt 解析器通常采用递归下降的方式实现，但它巧妙地结合了递归和循环，是一种递归为主、循环为辅的混合方法。

#### 核心实现结构 基本框架（递归下降）

```python
def parse_expression(self, precedence=0):
    # 递归解析左侧表达式
    left = self.parse_prefix()
    
    # 循环处理中缀操作符
    while self.current_token and precedence < self.get_precedence():
        left = self.parse_infix(left)
    
    return left
```

#### 详细解析过程

1. 递归部分：parse_prefix()

处理前缀表达式（字面量、前缀操作符等）：

```python
def parse_prefix(self):
    token = self.current_token
    
    if token.type == NUMBER:
        return self.parse_number()
    elif token.type == IDENTIFIER:
        return self.parse_identifier()
    elif token.type == LPAREN:
        return self.parse_group()
    elif token.type == MINUS:  # 前缀负号
        return self.parse_prefix_operator()
    # 递归调用
```

2. 循环部分：while 循环

处理中缀和后缀操作符：

```python
def parse_expression(self, precedence=0):
    left = self.parse_prefix()
    
    # 循环处理连续的中缀操作符
    while True:
        token = self.current_token
        if not token:
            break
            
        # 获取当前操作符的优先级
        current_precedence = self.get_precedence(token)
        if current_precedence <= precedence:
            break
            
        # 递归调用（结合了递归）
        left = self.parse_infix(left, token)
    
    return left
```

#### 递归的必要性

处理嵌套结构：括号、函数调用等需要递归

处理前缀操作符：-a + b 需要递归解析 -a

处理右结合操作符：a ^ b ^ c 需要递归控制优先级

#### 完整示例实现

```python
class PrattParser:
    def __init__(self, tokens):
        self.tokens = tokens
        self.position = 0
    
    @property
    def current_token(self):
        if self.position < len(self.tokens):
            return self.tokens[self.position]
        return None
    
    def advance(self):
        self.position += 1
    
    def get_precedence(self, token=None):
        if token is None:
            token = self.current_token
        
        precedences = {
            '+': 10, '-': 10,      # 低优先级
            '*': 20, '/': 20,      # 高优先级
            '^': 30,               # 更高优先级（右结合）
            '=': 5,                # 非常低优先级
        }
        return precedences.get(token.value, 0)
    
    def parse_expression(self, precedence=0):
        """核心递归函数"""
        # 1. 递归解析左侧（前缀表达式）
        left = self.parse_prefix()
        
        # 2. 循环处理中缀操作符
        while self.current_token and precedence < self.get_precedence():
            left = self.parse_infix(left)
        
        return left
    
    def parse_prefix(self):
        """递归处理前缀表达式"""
        token = self.current_token
        self.advance()
        
        if token.type == 'NUMBER':
            return NumberNode(token.value)
        elif token.type == 'IDENTIFIER':
            return IdentifierNode(token.value)
        elif token.value == '(':
            # 递归调用：解析括号内的表达式
            expr = self.parse_expression()
            if self.current_token.value != ')':
                raise SyntaxError("Expected ')'")
            self.advance()
            return expr
        elif token.value == '-':  # 前缀负号
            # 递归：解析右侧表达式
            right = self.parse_expression(30)  # 高优先级
            return PrefixOpNode('-', right)
        
        raise SyntaxError(f"Unexpected token: {token}")
    
    def parse_infix(self, left):
        """处理中缀操作符（结合递归和循环）"""
        token = self.current_token
        self.advance()
        
        # 获取结合性：大多数左结合，指数右结合
        token_precedence = self.get_precedence(token)
        if token.value == '^':  # 右结合
            right_precedence = token_precedence - 1
        else:  # 左结合
            right_precedence = token_precedence
        
        # 递归调用：解析右侧表达式
        right = self.parse_expression(right_precedence)
        
        return InfixOpNode(left, token.value, right)
```
