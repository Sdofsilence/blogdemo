# 最长回文子串

*寻找字符串中的**最长回文子串**是一个经典问题，常见的解决方法包括动态规划和中心扩展法。以下是两种高效的实现方式：*

## *方法 1: 中心扩展法*

*此方法通过从每个字符或字符间隙向外扩展，找到所有可能的回文子串。*

***步骤：***

1. *遍历字符串的每个字符位置，分别以单个字符和两个字符为中心进行扩展。*
2. *在扩展过程中，检查左右字符是否相等，记录最长的回文子串。*
3. *返回最长的回文子串。*

## *方法 2: 动态规划*

*动态规划通过记录子串是否为回文来优化计算。*

***步骤：***

1. 定义一个二维布尔数组 dp，其中`dp [ *i* ][ *j* ]`表示子串`s[ *i* : *j* +1]`是否为回文。
2. 初始化长度为 1 的子串为回文。
3. 根据状态转移方程更新 dp： 如果`*s* [ *i* ] ==  *s* [ *j* ]`且`*dp* [ *i* +1][ *j-1* ]`为真，则`*dp* [ *i* ][ *j* ]`为真。
4. 找到最长的回文子串。*

### *总结*

* **中心扩展法**适合快速实现，时间复杂度为 **O(n²)** ，空间复杂度为 **O(1)**。
* **动态规划**适合需要记录更多状态的场景，时间复杂度为**O(n²)**，空间复杂度为 **O (n²)**。
