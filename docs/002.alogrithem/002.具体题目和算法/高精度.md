# 大数加减乘除

## 加减用模拟

```
//反向存储 例如1234,存在vector<int8_t> {4,3,2,1}
vector<uint8_t> a;
vector<uint8_t> b;

//加法：从左到右加起来放到vectro<int8_t > c中，考虑计算进位值 ，需要考虑符号位，相同不变，否则符号位为绝对值大的，值为绝对值大的减绝对值小的（依赖于无符号减法）
//被减数符号为-则转化为加法，否则分两种情况（正数减正数（需要考虑绝对值大小得到符号位），负数减正数（符号位不变，绝对值相加））
```

## 乘法和除法

* 乘法用算式模拟有一个规律就是固定的第一个数的i位*第二个数的j位=结果i+j位的值，进位值写入结果i+j+1位(或者最后统一处理每一位上大于10的值)
* 除法分为两种：除数为整数（也用算式模拟），除数为大数（用减法代替除法）
