## 后缀数组的应用

* [题单：kuangbin后缀数组题单小结](https://www.nowcoder.com/discuss/353148645056913408)
* 最长公共前缀（lcp）:利用h数组，快速求解任意两个后缀的lcp.
* 最长重复子串：通过h数组找到最大的h[i],对应的后缀区间就是最长重复子串。（这个问题也可以用rabin-karp（字符串滚动哈希）+长度二分求解 推荐）

## 刷题手动实现的选择

* 最优的时间是O(n)的，但是只需要理解倍增法（计算rank数组）加基数排序实现的nlogn的计算sa数组。好理解和实现。

```c++
#include<string>
#include<vector>
#include<array>
#include<cmath>
#include<numeric>
using namespace std;

vector<int> make_suffix_array(string &s) {
    constexpr int N = 1e6;
    array<int,N> cnt;
    int len = s.size();
    vector<int> sa(len+1),sa2(len+1),rk(2*len+1,0),rk2(2*len+1,0);//sa和rank数组的下标为1-n

    for(auto& i : cnt) i = 0;
    for(int i = 1; i <= len; ++i) ++cnt[rk[i] = s[i-1]];//第一遍rk就是s[i],因为值相同排名就相同
    inclusive_scan(cnt.begin(),cnt.end(),cnt.begin());
    for(int i = len; i >= 1; --i) sa[cnt[s[i-1]]--] = i;//下标从1开始  sa的值是排名
    //长度为1时候的sa数组转换rank数组
    rk2 = rk;
    for(int i = 1,p = 0; i <= len; ++i){
        if(rk2[sa[i]] == rk2[sa[i-1]]){//rk2 == rk2 则排名相同sa数组就是按照排名 i = 1...n 排序的所以递增sa[i],如果排名相同则p不变
            rk[sa[i]] = p;
        }else{
            rk[sa[i]] = ++p;
        }
    }

    int w = 1;
    while(w <= len){
        //对第二关键字排序
        sa2 = sa;
        for(auto& i : cnt) i = 0;
        for(int i = 1; i <= len; ++i)  ++cnt[rk[sa[i]+w]];
        inclusive_scan(cnt.begin(),cnt.end(),cnt.begin());
        for (int i = len; i >= 1; --i) sa[cnt[rk[sa2[i]+w]]--] = sa2[i];
        //对第一关键字排序
        sa2 = sa;
        for(auto& i : cnt) i = 0;
        for(int i = 1; i <= len; ++i) ++cnt[rk[sa[i]]];
        inclusive_scan(cnt.begin(),cnt.end(),cnt.begin());
        for (int i = len; i >= 1; --i) sa[cnt[rk[sa2[i]]]--] = sa2[i];
        //从sa数组获取rank数组
        int p = 0;
        for(int i = 1; i <= len; ++i){
            if(rk2[sa[i]] == rk2[sa[i-1]] && rk2[sa[i]+w] == rk2[sa[i-1]+w])
                rk[sa[i]] = p;
            else{
                rk[sa[i]] = ++p;
            }
        }
        if(p == len) break;//已经排好序
        w <<= 1;
    }
    
    return sa;
}
```
